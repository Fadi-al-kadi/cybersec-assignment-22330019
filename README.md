# Cybersecurity Malware Analysis Assignment

Student ID: 22330019

This repository contains the results of a malware analysis assignment performed in a controlled FLARE VM environment.

The analysis includes:

- Static analysis of the malware sample
- Dynamic behavior observation
- Network traffic analysis
- Documentation of findings

The malware sample appears to behave similarly to ransomware based on file access patterns, encryption API usage, and targeted document file extensions.

Tools used:
- FLARE VM
- CFF Explorer
- Strings
- FLOSS
- Process Monitor
- Wireshark
- ProcDOT

All analysis was conducted in an isolated virtual machine environment.
